# Arya Kulkarni - Portfolio Website

## Setup Instructions

1. **Add your profile photo**: Save your profile image as `profile.jpg` in this folder

2. **Open the website**: Simply open `index.html` in your browser

3. **Deploy to hosting** (optional):
   - **Firebase Hosting**: `firebase deploy`
   - **GitHub Pages**: Push to a GitHub repo and enable Pages
   - **Netlify**: Drag and drop the folder to netlify.com
